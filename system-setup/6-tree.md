# Install Tree

**Run:**

 `brew install tree`

Verify tree is installed by **running:**

 `tree --version`

Expected Output:

```
tree v1.8.0 (c) 1996 - 2018 by Steve Baker, Thomas Moore, Francesc Rocher, Florian Sesser, Kyosuke Tokoro
```
