#  Check Homebrew

Run:

`brew doctor`

**If** output is: 
```
Your system is ready to brew
``` 
**run:**

`brew update` 

 [Continue to next step](../tree/tree.md)

 ---

**Else** output is: 
```
-bash: brew: command not found
```
 
 **then** [click here to install Homebrew](../homebrew/homebrew.md)
