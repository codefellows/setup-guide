# live-server

This is a simulated website development server with live reload capability. This will allow you to see your website before deoploying.

## Installation

**Run:**

 `npm install -g live-server`

Verify live-server is installed by **running:**

 `live-server --version`

Expected Output:

`live-server 1.2.1`
